# Digital Image Processing Lab - COMP-342L

This repository contains lab exercises and materials for the Digital Image Processing course (COMP-342L).

## Structure

- **Lab 1-12**: Individual lab exercises and implementations
- **Lab Manuals**: Lab manuals and documentation
- **Lab Reports**: Lab reports and submissions

## Setup

1. Create and activate a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
# or
venv\Scripts\activate  # On Windows
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

## Usage

Navigate to individual lab folders for specific exercises and implementations.
