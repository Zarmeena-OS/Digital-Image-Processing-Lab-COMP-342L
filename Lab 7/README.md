# Lab 7 — Color preservation (CLAHE in HSV)

Apply CLAHE only to the V (value) channel in HSV; H and S unchanged to avoid color shift.

## Run

```bash
cd "Lab 7"
python color_preservation.py
```

## Output

- **`Lab 7/Lab7_color_preserved.png`** — Original RGB | CLAHE on V (color preserved).

## Files

- `color_preservation.py` — main script; `Content/` — optional images.
